## 담당 파트 소개




### 1. 회원가입  및 로그인

- 오어스를 이용한 회원가입과 로그인

  

### 2. 채팅 기능

- 아티스트와 팬 채팅기능
- 불량유저 차단
- 채팅 구독 기능




### 3. 매니지먼트 관리

- 아티스트 가입 및 관리
- 그룹(팀) 관리
- 굿즈 관리
- 현황 대시보드

  

### 4. 게시판

- 아티스트 전용 게시판
- 팬 전용 게시판
- 매니지먼트 공지란
- 팀에 소속된 아티스트 프로필

  

### 5. 굿즈 이커머스

- 상품 둘러보기
- 장바구니에 상품 넣기
- 장바구니에 넣은 상품 결제


  
### 6. 1대1 영상팬미팅

- 방 추가
- 대기방
- 미팅방
- 대기방 목록
